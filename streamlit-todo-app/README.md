# Streamlit To-Do App

This is a simple to-do list application built using Streamlit. It allows users to manage their tasks efficiently with an interactive interface.

## Features

- Add new tasks to your to-do list.
- Remove completed tasks.
- View all current tasks.

## Requirements

To run this application, you need to have Python installed on your machine. The required libraries are listed in the `requirements.txt` file.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd streamlit-todo-app
   ```

2. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Running the App

To start the Streamlit application, run the following command in your terminal:
```
streamlit run app.py
```

This will open the application in your default web browser.

## Usage

- Use the input field to add new tasks.
- Click on the task to mark it as completed and remove it from the list.
